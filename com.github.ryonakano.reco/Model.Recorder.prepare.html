public bool prepare (string dst_path, SourceID source, ChannelID channel, FormatID format, string? meta_author, DateTime? meta_record_dt)

Create a pipeline and add elements to it

This changes state of this from Idle to Ready if succeeds

Parameters:

dst_path

destination path where to save recording

source

information of which type of a device records from

channel

number of channels

format

file format that is encoded to

meta_author

artist name that will be set to metadata as value of "Artist". Set null for no metadata

meta_record_dt

date & time that will be set to metadata as value of "Year". Set null for no metadata

Returns:

true if succeeds, false otherwise
