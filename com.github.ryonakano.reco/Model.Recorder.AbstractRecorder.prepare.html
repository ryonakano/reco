prepare

Description:

public abstract bool prepare (Pipeline pipeline, Element mixer, Element sink)

Prepare for recording.
All elements created in this method should be added to #pipeline using Gst.Bin.add and linked to #mixer or #sink appropriately using Gst.Pad.link.

|-- #pipeline --------------------------------------------------------------------|
|           |------------|  |-----------------|                    |------------| |
|  (snip) --|   #mixer   |--| format-specific |-- (snip, if any) --|   #sink    | |
|         --|            |--|     element     |--                --|            | |
|           |------------|  |-----------------|                    |------------| |
|---------------------------------------------------------------------------------|
                         <--------- scope of this method ---------->

NOTE:
Overridden methods should add at least one element that inherits Gst.TagSetter to #pipeline for metadata.
See Manager.RecordManager.add_metadata for details.

Parameters:

pipeline

pipeline that holds all elements necessary for recording.

mixer

audiomixer that precedes all elements created in this method.

sink

filesink that succeeds to all elements created in this method.

Returns:

true if succeeds, false otherwise.
