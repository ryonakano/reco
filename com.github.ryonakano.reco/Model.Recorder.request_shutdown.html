public bool request_shutdown ()

Requests for graceful shutdown

Requests to stop ongoing recording safely if such exists

Returns:

true if shutdown completed; state is set to RecordState.IDLE .
false otherwise; the caller must wait until either one of record_ok or record_err signals being emitted. state is set to RecordState.IDLE after either signal is emitted
