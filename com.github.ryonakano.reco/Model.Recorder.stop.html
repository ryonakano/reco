public bool stop ()

Stop recording

This changes state of this from Recording or Paused to Finalizing if succeeds

Note:

This method just send an end-of-stream event to an internal pipeline; the return value does not indicate whether the pipeline handles the event successfully.
Instead, record_err is thrown if recording completed successfully. Connect to it before calling this method

Returns:

true if succeeds, false otherwise
